Linux Path Quick Notes

1. Absolute Path → Full path starting from root `/`
   Example: `/mnt/c/Users/Anshuman/Desktop`

2. Relative Path → Path relative to current directory
   Example: `cd project`

3. Current Directory → `.` means current folder
   Example: `./script.sh`

4. Parent Directory → `..` means one folder up
   Example: `cd ..`

5. Home Directory → `~` means user's home folder
   Example: `cd ~`
